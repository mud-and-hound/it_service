ส่วนที่ 02

ปุ่มตั้งค่าเสริมหน้าเว็บ

แถบด้านบนขวาของหน้าจอมีปุ่มลัดสำหรับปรับแต่งการใช้งาน — ตั้งค่าเฉพาะแต่ละบัญชี ไม่กระทบผู้อื่น

1

สถานะสีหลังเวลา

Connection Status

จุดสีข้างเวลาแสดงตลอดการใช้งาน บอกว่าข้อมูลเชื่อมต่อระบบแล้วหรือยัง — ดูรายละเอียดสีแต่ละแบบที่หัวข้อถัดไป

2

ปุ่มรีเฟรช

Refresh

กดเพื่อโหลดหน้าเว็บใหม่ ในกรณีที่ข้อมูลไม่ขึ้นหรือค้าง ดึงข้อมูลล่าสุดจากเซิร์ฟเวอร์

3

เปลี่ยนภาษา EN / TH

Language

สลับภาษาไทย ↔ อังกฤษ ในกรณีที่ไม่เข้าใจภาษาที่ระบบตั้งไว้ให้

4

ปุ่มแจ้งเตือน

Notifications

เช็คว่าคุณทำอะไรไปแล้วบ้าง สำหรับตรวจทานการทำงานของตัวเอง และรับการแจ้งเตือนจากระบบ

5

Home

หน้าหลัก

กดเพื่อเยี่ยมชมหน้าหลัก (Landing) ของเว็บไซต์ — เลือกแบรนด์หรือเข้าสู่ระบบส่วนอื่นได้

6

โหมดมืด

Dark Mode

สลับธีมของระบบเป็นสีเข้ม ถนอมสายตาเวลาใช้งานกลางคืน — ลองกดปุ่มพระจันทร์มุมขวาบนของคู่มือนี้ดูได้เลย

7

ออกจากระบบ

Log out

ส่วนที่ 02.1

สถานะสีของระบบ

จุดสีข้างเวลาบอกสถานะการเชื่อมต่อข้อมูลแบบเรียลไทม์ จำ 3 สีนี้ไว้เพื่อรู้ว่าระบบพร้อมใช้งานหรือไม่

สีฟ้าระบบกำลังรีเฟรชหรืออัปเดตข้อมูลอยู่ — รอสักครู่ให้โหลดเสร็จ
สีเขียวระบบใช้งานได้ปกติ ข้อมูลเชื่อมต่อเรียบร้อย ทำงานต่อได้เลย
สีแดงระบบไม่ได้เชื่อมต่อข้อมูล — ลองกดปุ่ม Refresh หรือเช็คอินเทอร์เน็ต
ส่วนที่ 03

สถานะงาน (Ticket)

ทุกงานจะไหลผ่านสถานะเหล่านี้ตามลำดับ — Admin และช่างเปลี่ยนสถานะได้ ทำให้ทุกคนเห็นความคืบหน้าตรงกัน

รอดำเนินการ→ กำลังทำ→ รออะไหล่รอซัพซ่อม→ ตรวจงาน→ เสร็จสิ้น

รอดำเนินการ

Pending

ร้านเพิ่งแจ้งปัญหาเข้ามา ยังไม่มีช่างรับงาน รอ Admin มอบหมาย

กำลังทำ

In Progress

ช่างรับงานและกำลังดำเนินการซ่อมที่หน้างาน

รออะไหล่

Waiting Parts

งานหยุดชั่วคราวเพราะรออะไหล่/อุปกรณ์ ช่างระบุเหตุผลไว้ให้ Admin เห็น

รอซัพดำเนินการซ่อม

Waiting Vendor

ต้องส่งช่างภายนอก/ผู้ผลิตมาซ่อม รอคิวดำเนินการจากซัพพลายเออร์

ตรวจงาน

Review

ช่างส่งงานแล้ว รอ Admin ตรวจรับ ดูรายละเอียดงาน + รูปก่อนปิด

เสร็จสิ้น

Done

Admin ตรวจรับและปิดงานเรียบร้อย ระบบแจ้งผู้แจ้งผ่าน LINE (ถ้ามี)

รออะไหล่ และ รอซัพซ่อม เป็นสถานะพักงานชั่วคราว เมื่อช่างพร้อมทำต่อ กด "ดำเนินการต่อ" งานจะกลับสู่ "กำลังทำ" โดยไม่รีเซ็ตเวลาที่ทำไปแล้ว
