หน้าที่ 01

Dashboard — หน้าเริ่มต้น

หน้าเพจแรกเป็น Dashboard เพื่อให้ช่างรับทราบว่างานที่ได้รับมีกี่งาน ตรวจสอบงานกี่งาน เสร็จแล้วกี่งาน และทั้งหมดมีเท่าไหร่ เหมือนมองเห็นภาพรวมของบัญชีว่าเราทำไปบ้างแล้วหรือยัง

  • งานที่ได้รับ — เมื่อกดจะเด้งไปที่งานตัวเองที่ได้รับ
  • รออะไหล่ — เมื่อกดจะเด้งงานที่สถานะ รออะไหล่
  • รอซัพ — เมื่อกดจะเด้งงานที่สถานะ รอซัพ
  • การ์ดสรุปตัวเลข — งานที่รับ / ตรวจงาน / เสร็จแล้ว / ทั้งหมด
  • กราฟงาน 7 วัน — จำนวนงานย้อนหลัง 7 วันที่ผ่านมา
หน้าที่ 02

แถบเมนู 7 รายการ

กดปุ่มเมนู (☰) มุมซ้ายบน จะเห็นเมนูทั้งหมด 7 รายการให้กด

1 · Dashboard

ภาพรวมที่แสดงว่างานของคุณดำเนินการไปถึงไหนแล้ว

2 · งานของฉัน

งานที่มีและทำอยู่ หรือได้รับงานมาแต่ยังไม่ได้กดเริ่มงาน

3 · ตรวจงาน

งานที่ส่งไปแล้วเหลือให้ Admin ตรวจสอบข้อมูล

4 · เสร็จสิ้น

งานที่ Admin ตรวจงานแล้วกดจบงานให้แล้ว

5 · ประวัติ

ระบบเก็บข้อมูลว่าคุณทำงานอะไรไปบ้างหรืออัปเดตอะไรไป

6 · ตั้งค่า

ตั้งค่าข้อมูล รับแจ้งเตือน หรือ Dark Mode / ภาษาไทย

7 · ฉุกเฉิน

เบอร์โทรติดต่อฉุกเฉิน กดแล้วโทรทันทีหลังยืนยันการโทร

หน้าที่ 03

งานของฉัน

งานที่คุณมีและทำอยู่ หรือได้รับงานมาแต่ยังไม่ได้กดเริ่มงาน

  • จำนวนต่อหน้า — เลือกแสดงกี่รายการบนหน้าเว็บ
  • รายการ Ticket — กดรับงาน / เริ่มงาน / นำทาง หรือกดติดขัด รอซัพ รออะไหล่
  • ช่องค้นหา — ID Ticket / ประเภท / สาขา
  • ปุ่ม Refresh — โหลดเว็บใหม่กรณีข้อมูลไม่ขึ้น
หน้าที่ 04

ตรวจงาน

งานที่ส่งไปแล้วเหลือให้ Admin ตรวจสอบข้อมูล

  • จำนวนต่อหน้า — เลือกแสดงกี่รายการบนหน้าเว็บ
  • รายการ Ticket — งานที่เราส่งให้ Admin ตรวจสอบ
  • ช่องค้นหา — ID Ticket / ประเภท / สาขา
  • ปุ่ม Refresh — โหลดเว็บใหม่กรณีข้อมูลไม่ขึ้น
หน้าที่ 05

เสร็จสิ้น

งานที่ Admin ตรวจงานแล้วกดจบงานให้แล้ว

  • จำนวนต่อหน้า — เลือกแสดงกี่รายการบนหน้าเว็บ
  • รายการ Ticket — งานที่ Admin ตรวจสอบแล้วกดจบงานให้
  • ช่องค้นหา — ID Ticket / ประเภท / สาขา
  • ปุ่ม Refresh — โหลดเว็บใหม่กรณีข้อมูลไม่ขึ้น
หน้าที่ 06

ประวัติ

ระบบเก็บข้อมูลว่าคุณทำงานอะไรไปบ้างหรืออัปเดตอะไรไป

  • จำนวนต่อหน้า — เลือกแสดงกี่รายการบนหน้าเว็บ
  • รายการ Ticket — ประวัติงานทั้งหมดที่ทำไป
  • ช่องค้นหา — ID Ticket / ประเภท / สาขา
  • ปุ่ม Refresh — โหลดเว็บใหม่กรณีข้อมูลไม่ขึ้น
  • ช่วงวันที่ — ค้นหาช่วงระหว่างวันที่ เพื่อหางานหรือ Ticket
  • เลือกแบรนด์ — Filter (กรณีช่างรับหลายแบรนด์)
หน้าที่ 07

ตั้งค่า

ตั้งค่าข้อมูล รับแจ้งเตือน หรือ Dark Mode / ภาษาไทย

  • แจ้งเตือนงานใหม่ — ตั้งว่าต้องการให้แจ้งเตือนหรือไม่
  • GPS Auto-track — หากปิดไปแล้ว จะไม่สามารถกดรับงาน/จบงานได้
  • Dark Mode — ธีมเว็บสีดำกับสว่าง
  • ภาษา EN/TH — ตั้งค่าภาษา

สำคัญ: ปุ่ม GPS Auto-track ต้องเปิดไว้เสมอ ถ้าปิดจะกดรับงาน/จบงานไม่ได้ เพราะระบบต้องใช้ตำแหน่งยืนยันการทำงาน

ตัวอย่างโหมดมืด (Dark Mode)

หน้าที่ 08

ฉุกเฉิน

เบอร์โทรสำหรับการติดต่อฉุกเฉิน กดแล้วจะโทรทันทีหลังกดยืนยันการโทร ระบบจะนำมือถือคุณโทรทันที

  • ช่องค้นหา — ค้นหาเบอร์หรือรายการฉุกเฉิน
  • หัวข้อหมวดหมู่ — เหตุด่วน / การแพทย์ / สาธารณูปโภค กดเพื่อดูเบอร์
  • ปุ่มกดเบอร์ — เมื่อกดไปแล้วจะโทรทันที (เช่น PEA 1129, MEA 1130)
