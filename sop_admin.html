Overview — ภาพรวมระบบ

ดูรายการทั้งหมดที่เกิดขึ้นตามรายวัน / เดือน / ปี แสดงเป็นกราฟเส้นและกราฟวงกลม กดคลิกดูรายละเอียดตามกราฟได้เลย

/admin · Overview
กดเพื่อขยาย

การ์ดสรุปด้านบนขวาแยกตามสถานะ: ทั้งหมด · รอดำเนินการ · กำลังทำ · รออะไหล่ · รอซัพ · ตรวจงาน · เสร็จสิ้น · ยกเลิก

กดการ์ดเพื่อดูรายการ POPUP

กดที่การ์ดสถานะใดก็ได้ ระบบจะเด้งรายการงานเฉพาะสถานะนั้นขึ้นมาให้กดดูรายละเอียดต่อ

ส่วนที่ 01.2

Ticket — ตั๋วงาน

ใช้ดู Ticket หรือทำรายการ — สร้างงาน / สั่งงานช่าง / ปรับสถานะงาน / จบงาน

หน้ารายละเอียด Ticket DETAIL

กดที่ Ticket เพื่อดูข้อมูล จะเห็นปุ่มปฏิบัติการและข้อมูลผู้แจ้ง

กดเพื่อขยาย
  • Assign — เลือกบุคคลเพื่อมอบหมายงานให้ช่าง
  • Close Ticket — กดปิดงานหรือจบงาน
  • Note — แสดงความคิดเห็นหรือเขียนเพิ่มเติม
  • Delete — ลบ Ticket
  • Status — สถานะงานที่แสดงปัจจุบัน (เช่น ตรวจงาน)
หน้าตาราง Ticket ทั้งหมด TABLE
/admin · All Tickets
กดเพื่อขยาย
  • ข้อมูล Ticket — เลข Ticket / ประเภท / แบรนด์ / สาขา / ช่าง / สถานะ / วันที่
  • ช่องค้นหา — พิมพ์หาเลข Ticket หรือข้อมูลที่ต้องการ
  • Filter สถานะ — กรองดูเฉพาะสถานะที่เลือก
  • เลือกแบรนด์ — กรองตามแบรนด์
  • เลือกช่าง — กรองดูเฉพาะช่างคนนั้น
  • สร้าง Ticket — สร้างงานใหม่เอง กรณี Admin รับงานแยกนอกเหนือจาก Website Report
ส่วนที่ 01.3

User — ผู้ใช้

สร้างบัญชี / ลบบัญชี / แก้ไขบัญชี และจัดการสิทธิ์ผู้ใช้ในระบบ

/admin · Users & Reporters
กดเพื่อขยาย
  • เลขรหัส / ชื่อ ID User — ข้อมูลประจำตัวผู้ใช้
  • จำนวนต่อหน้า — เลือกแสดง 5 / 10 หรือมากกว่า
  • ค้นหารหัสพนักงาน — หาจากรหัส/ชื่อ ID
  • ค้นหาชื่อ-นามสกุล — หาจากชื่อที่สร้างไว้
  • เลือกแบรนด์ — กรองเฉพาะแบรนด์
  • คู่มือ Role — ดูตำแหน่งของตนเองและสิทธิ์การสร้าง User
  • Add User — สร้างบัญชีผู้ใช้ใหม่
  • เลือก Role — ค้นหาตำแหน่ง Role
ส่วนที่ 01.4

GPS Map — แผนที่ GPS

ติดตามตำแหน่งของทีมช่างได้ทุกคนที่อยู่ในแบรนด์นั้น แบบเรียลไทม์

/admin · GPS Map
กดเพื่อขยาย
  • ข้อมูลช่างออนไลน์ — ดูได้ว่าช่างนั้นรับ Ticket อะไรอยู่
  • ตำแหน่งปัจจุบัน — จุดที่ช่างใช้งานเว็บ Engineer อยู่
  • ปุ่ม +/- ซูม — ซูมเข้าออกใน Map หรือ Scroll Mouse เลื่อนเข้าออก
  • อัปเดตล่าสุด — ข้อมูล Map อัปเดตล่าสุดเมื่อไหร่
ส่วนที่ 01.5

Branches — สาขา

ดูรายการสาขา ตำแหน่ง เบอร์โทร และแก้ไขข้อมูลสาขาได้

/admin · Branches
กดเพื่อขยาย
  • ข้อมูลสาขา — Code / Brand / ชื่อสาขา / เบอร์โทร / สถานที่ / สถานะ / ปุ่มแก้ไข
  • ช่องค้นหา — หาสาขาหรือชื่อสาขา

ปุ่มตั้งค่าเสริมหน้าเว็บ

แถบด้านบนขวามีปุ่มลัดสำหรับปรับแต่งการใช้งาน — ตั้งค่าเฉพาะแต่ละบัญชี ไม่กระทบผู้อื่น

สถานะสีหลังเวลา

Connection Status

จุดสีข้างเวลาแสดงตลอดการใช้งาน บอกว่าข้อมูลเชื่อมต่อระบบแล้วหรือยัง

2

ปุ่มรีเฟรช

Refresh

กดเพื่อโหลดหน้าเว็บใหม่ ในกรณีที่ข้อมูลไม่ขึ้นหรือค้าง

3

เปลี่ยนภาษา EN / TH

Language

สลับภาษาไทย ↔ อังกฤษ ในกรณีที่ไม่เข้าใจภาษาที่ระบบตั้งไว้ให้

4

ปุ่มแจ้งเตือน

Notifications

สถานะสีของระบบ

จุดสีข้างเวลาบอกสถานะการเชื่อมต่อข้อมูลแบบเรียลไทม์

สีฟ้าระบบกำลังรีเฟรชหรืออัปเดตข้อมูลอยู่ — รอสักครู่
สีเขียวระบบใช้งานได้ปกติ ข้อมูลเชื่อมต่อเรียบร้อย
สีแดงระบบไม่ได้เชื่อมต่อข้อมูล — ลองกด Refresh หรือเช็คอินเทอร์เน็ต
