ภาพรวม

หลักการทำงาน

Flow การแจ้งซ่อมตั้งแต่กรอกฟอร์มจนถึงช่างเริ่มงาน

กรอกแบบฟอร์ม — คุณกรอกแบบฟอร์มในระบบ ระบบจะแจ้งไปที่ Admin ทันที

ติดตามสถานะ — กดติดตามสถานะของ Ticket ได้ว่าถึงขั้นตอนไหนแล้ว

Admin มอบหมายงาน — เมื่อ Admin รับงานแล้วจะสั่งงาน (Assign) ให้ช่างไปหน้างาน

ช่างเริ่มงาน — ระบบจะเปลี่ยนแปลงข้อมูลอัปเดตลงในหน้าติดตาม Ticket

หน้าที่ 01

หน้าเข้าสู่ระบบ

หน้าเพจแรกเป็นหน้า Login ให้กรอกรหัสพนักงาน และรหัสผ่านคือ 4 ตัวท้ายของรหัสพนักงาน

รหัสผ่าน: ใช้ 4 ตัวท้ายของรหัสพนักงาน — หากเข้าไม่ได้ ให้ลองใส่รหัสพนักงานเต็มเป็นรหัสผ่านเหมือนกัน

หน้าที่ 02

แถบด้านบน

ปุ่มลัดมุมขวาบนสำหรับปรับแต่งการใช้งานและออกจากระบบ

  • R4790 — ชื่อ User ของท่านที่ล็อกอินอยู่
  • EN / TH — ปุ่มเปลี่ยนภาษา
  • Dark — ปุ่มเปิด/ปิดโหมดธีมของระบบ สีมืดกับสว่าง
  • ออกจากระบบ — กดแล้วออกไปหน้า Login
  • Home — กดแล้วเด้งไปหน้า Landing Web page ของระบบ
  • แถบแท็บ — สลับระหว่าง "แจ้งปัญหา" และ "ติดตาม Ticket"
หน้าที่ 03

แจ้งปัญหา

กรอกฟอร์มเพื่อแจ้งปัญหา ระบบจะสร้าง Ticket และส่งให้ทีม IT ทันที

กดเพื่อขยาย
ช่องกรอกข้อมูล FORM

รหัสสาขา

เลือกสาขาที่พบปัญหาหรือต้องการแจ้ง

ชื่อผู้แจ้ง

ชื่อจะกรอกเองเมื่อผู้ใช้เข้า Login หากไม่ขึ้นสามารถกรอกได้

เบอร์โทร

กรอกเบอร์โทรของท่านหรือเบอร์ร้าน / ผู้จัดการ

ประเภทปัญหา

เลือกหัวข้อสำหรับการแจ้งซ่อม หากไม่ตรงกับสิ่งที่เจอ กด "อื่นๆ" เพิ่มเติมได้

รายละเอียด

กรอกข้อมูลต้อง 10 ตัวอักษรขึ้นไป เพราะหากน้อยไป ระบบจะส่งไม่ได้

สถานที่

กรอกรายละเอียดคร่าวๆ เพื่อให้ช่างที่มาสาขาสังเกตหรือพบร้านได้ง่ายขึ้น ลดปัญหาการโทรหาร้านโดยไม่จำเป็น

รูปภาพ

แนบรูปได้ มากสุด 3 รูป รูปจะไปแสดงในระบบให้ช่างดู

ส่งรายงาน

ส่งข้อมูลฟอร์มนี้ไปที่ทีมที่รอแก้ไข

เคล็ดลับ: ยิ่งกรอก สถานที่ ละเอียด (ชั้น / โซน / ตำแหน่งเครื่อง) ช่างยิ่งหาเจอเร็ว งานเสร็จไวขึ้น

หน้าที่ 04

ติดตาม Ticket

ติดตามข้อมูล Ticket ที่เคยแจ้งไว้ ดูสถานะและความคืบหน้าได้ตลอดเวลา

กดเพื่อขยาย
  • ค้นหา Ticket ID — พิมพ์หารหัสหรือข้อมูลที่เกี่ยวข้อง ระบบจะแสดง Ticket ที่เกี่ยวข้องให้ทันที
  • ปุ่มค้นหา — เมื่อกรอกข้อมูลแล้วกดค้นหาเพื่อหาข้อมูล
  • ดูตามสาขา — เลือกสาขาที่ต้องการดู
  • ทุกสถานะ — เลือกกรองตามสถานะได้ในกรณีที่อยากดูเพิ่มเติม
  • จากวันที่ / ถึงวันที่ — กำหนดระยะวันที่เพื่อดู Ticket ตามช่วงเวลาที่ระบุ
  • แสดง — เลือกจำนวนรายการต่อหน้า 5 / 10 / 20 / 50 / 100
  • รายการที่แสดง — Ticket ที่แสดงตามที่เรา Filter ไว้ กดดูรายละเอียดได้
